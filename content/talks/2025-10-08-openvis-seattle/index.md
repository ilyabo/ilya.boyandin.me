# SQLRooms: Build powerful analytics apps with DuckDB in browser

I gave this talk at the [Open Visualization Collaborator Summit 2025](https://deck.gl/events/seattle-summit-2025/) in Seattle, part of the [Open Visualization](https://www.openvisualization.org/) community within the OpenJS Foundation.

In the talk I introduced [SQLRooms](/p/sqlrooms), the modular React framework for building analytics applications powered by [DuckDB](https://duckdb.org/) either entirely in the browser or with a backend when needed. SQLRooms provides a foundation and set of building blocks for creating data analytics applications that combine a SQL query engine, visualization tools, state management, AI integration, and UI components into a single cohesive toolkit.

The main idea behind SQLRooms is that powerful analytics apps do not have to start with a heavyweight backend architecture. With DuckDB running in the browser, developers can build fast, private, and highly capable data tools that stay local by default while still supporting richer deployment options when datasets or collaboration requirements grow.
